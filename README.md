
MACTI_LIB es una biblioteca desarrollada en el marco de dos proyectos del programa UNAM-DGAPA-PAPIME (PE101019 y PE101922) y con apoyo del concurso InnovaUNAM para la Enseñanza y Aprendizaje de Contenidos Curriculares Prácticos en Ciencias y Humanidades a Distancia 2021. Contiene herramientas para evaluación de ejercicios numéricos, para la visualización de datos y para la simulación de fenómenos geofísicos simples. 

Autores principales: Luis M. de la Cruz Salas, Miguel Angel Pérez León
